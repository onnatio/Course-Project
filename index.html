First, I load the caret package and the data file named "pml-training":

library(caret)
training<-read.csv("C:/Users/Antonio/Documents/R/pml-training.csv",header=TRUE)

I split the data set into training and testing subsets, named "train" and "test", respectively.

set.seed(8090)
inTrain<-createDataPartition(y=training$classe,p=0.75,list=FALSE)
train<-training[inTrain,]
test<-training[-inTrain,]
dim(training)
dim(train)
dim(test)

In order to predict the variable "Classe", I use the Random Forest algorithm in the package "caret".I do not include in the algorithm variables with too many NA's or variables which do not give valuable information (like usernames, etc.).

mod_rf = train(classe~raw_timestamp_part_1+raw_timestamp_part_2
              +new_window+num_window+roll_belt+pitch_belt
              +yaw_belt+total_accel_belt+gyros_belt_x+gyros_belt_y
              +gyros_belt_z+accel_belt_x+accel_belt_y+accel_belt_z
              +magnet_belt_x+magnet_belt_y+magnet_belt_z+roll_arm
              +pitch_arm+yaw_arm+total_accel_arm+gyros_arm_x
              +gyros_arm_y+gyros_arm_z+accel_arm_x+accel_arm_y+accel_arm_z
              +magnet_arm_x+magnet_arm_y+magnet_arm_z+roll_dumbbell+pitch_dumbbell
              +yaw_dumbbell, method = "rf", data = train)

The results are shown here; we see that the estimate of the error rate is very low (0,1%); I also show the importance of the variables:

print(mod_rf$finalModel)
varImport<-varImp(mod_rf)
varImport

Next, I use the test subset to validate the model. I build the Confusion Matrix and we can see that the accuracy is very high, 0,998.

preds<-predict(mod_rf,newdata=test)
confusionMatrix(preds,test$classe)

Finally, I use the model to predict the values of the variable "Classe" of a new set of data named "pml-testing". The 20 predictions are shown below:

testing<-read.csv("C:/Users/Antonio/Documents/R/pml-testing.csv",header=TRUE)
testing_preds<-predict(mod_rf,newdata=testing)
testing_preds
